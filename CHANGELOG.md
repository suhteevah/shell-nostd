# Changelog

## 0.1.0 (2026-04-03)

- Initial release
- 45+ builtin commands (ls, cd, cat, cp, mv, rm, mkdir, rmdir, echo, grep, find, etc.)
- Pipeline support (pipes between commands)
- Environment variable expansion
- Shell scripting (if/else/while/for, functions, variables)
- Command parser with quoting, escaping, redirections
- AI-native integration hooks (ask, agent commands)
- Customizable prompt with working directory display
